# ParkNidus - Sistema de Control de Parqueadero

**Autora**: Giseella Sanchez

Sistema web moderno de control de parqueadero con:
- Dashboard cyberpunk neon
- Gestión de espacios, vehículos, tarifas
- Reportes e ingresos
- Autenticación completa
- Base de datos Supabase

## 🚀 Quick Start

1. Copia `.env.local.example` → `.env.local` con tus credenciales Supabase
2. `npm install`
3. `npm run dev`
4. http://localhost:3000

**Credenciales demo**:
- admin@parking.com / admin123
- operario@parking.com / oper123

## 🎨 Tema Neon Pulse
Paleta cyberpunk: Cyan eléctrico, violeta profundo, lima eléctrica.

## 📊 Supabase
Ejecuta `scripts/supabase-schema.sql` en tu dashboard.
